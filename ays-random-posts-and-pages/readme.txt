### WordPress Random Posts and Pages

* [WordPress Random Posts plugin pricing](https://ays-pro.com/wordpress/random-posts-and-pages)
* [Random Posts Free Demo](https://plugins.ays-demo.com/random-posts-and-pages-free-demo/)
* [Random Posts Pro Demo](https://plugins.ays-demo.com/random-posts-and-pages-pro-demo/)

https://www.youtube.com/watch?v=nOkwIB50ah0

[WordPress Random Post plugin](https://ays-pro.com/wordpress/random-posts-and-pages) serves the purpose of creating internal links and encouraging visitor engagement on your website. 
It allows you to showcase a dynamic display of random links, changing with each interaction. Within a container, you can include an unlimited number of links and effortlessly modify them as needed. Use this plugin to add interactivity to your website, attracting a greater number of visitors, making them stay on your page, and increasing your SEO ranking.

 We give you chance to change widget’s parameters and customize your own style(color, width, background color, border color etc). 
The main advantage of this widget is random movement of random links and every time they are changing.
 The back end interface is user-friendly and you can easily navigate and change parameters.
 
== FEATURES ==
* Responsive design

== ADVANCED SETTINGS ==

* Ability to choose the animation type and specify the animation speed
* Set the box width and the box height
* Choose the box background color 
* Upload a background image for the container
* Add shadow effects to the container
* Choose a link text color and a link background color
* Add padding for links
* Select your desired text font
* Choose to display the featured image of a link already open or while hovering over it
* Set the link image height and its size
